# Sunset Bakery Website

A responsive bakery website built using HTML, CSS, and JavaScript.

## Features
- Home Page
- About Page
- Contact Form
- Business Hours Page
- Responsive Design
- Dynamic Footer Year using JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript
